# 🎮 AAA

사용된 에셋 패키지들은 라이선스 규정에 따라 제외되었기 때문에 현재 리포지터리에서 확인이 불가합니다.

</br>

## 🖥️ 프로젝트 소개

같은건 없고 무작정 이동, 전투 구현하고있습니다. 기획 전혀 없습니다.

굳이 말하자면 적 다 쓰러트리면 다음 스테이지 넘어가고 뭐... 얼마나 오래 버티나..? 같은느낌.

메이플스토리의 무릉도장이라고 해야할까요. 만들 수 있을지도 모르겠음.

레퍼런스는 원신, 퍼니싱 그레이 레이븐, 니어 오토마타와 같이 회피기동이 있는 전투 만들어보려는 중 입니다.

[진행사항은 여기서 보실 수 있습니다. (별거없습니다 보지마세요)](https://www.youtube.com/@MyCatJazz)

## 📝 Code Convention
```bash
◾ Variable : 카멜케이스 (ex. int level)
◾ Bool : is~, can~, has~ (ex. bool isPlay)
◾ Class, func : 파스칼 (ex. public void OnMove())
◾ Enum, 상수 : 전부 대문자 (ex. UNIT_AA, const int PI)
```


## ✍ Commit Convention Rule

```bash
Year-Month-Day [Filename] Add/Fix/Update/Move > Info

ex) 2024-04-26 [meleeAttack.cs] Add > 파일을 추가했습니다.
ex) 2024-04-26 [PlayerMovement.cs] Update > 각주를 추가했습니다.
```
